# 理解项目

在应用开发模式下，快速理解项目结构是高效开发的关键。

## 说明

在应用开发模式下，你已经获得了项目根目录地址，并且运行环境旁边已经有一个浏览器模拟器打开了该应用项目。

你可以随时操作模拟器，查看应用项目的运行状态和页面结构。

你可以使用 `emulator-navigate` 能力来获取当前模拟器的网页地址、刷新当前页面或跳转到指定页面。具体使用 `get-capability` 能力获取该能力详情。
